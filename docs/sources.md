# Поддерживаемые источники

Агент поддерживает парсинг материалов из следующих источников:

## 1. Habr.com

**Тип контента:** Технические статьи

**Формат URL:**
```
https://habr.com/ru/articles/[ID]/
```

**Пример:**
```
https://habr.com/ru/articles/984968/
```

**Что извлекается:**
- Заголовок статьи
- Автор
- Дата публикации
- Полный текст статьи (до 8000 символов)

---

## 2. GitHub.com

**Тип контента:** Документация из репозиториев

**Формат URL:**
```
https://github.com/[owner]/[repo]
```

**Пример:**
```
https://github.com/anthropics/anthropic-cookbook
```

**Что извлекается:**
- Название репозитория
- Владелец
- Описание
- README.md
- Дополнительные markdown файлы:
  - ARCHITECTURE.md
  - CONTRIBUTING.md
  - DEVELOPMENT.md
  - SETUP.md
  - INSTALL.md
  - USAGE.md
  - API.md
  - CHANGELOG.md
  - Все .md файлы из папки docs/
- Количество звёзд
- Основной язык программирования

**Особенности:**
- Использует GitHub API для поиска файлов
- Объединяет содержимое всех найденных файлов
- Максимальная длина: 16000 символов

---

## 3. InfoStart.ru

**Тип контента:** Статьи и публикации по 1С

**Форматы URL:**
```
https://infostart.ru/1c/articles/[ID]/
https://infostart.ru/public/[ID]/
```

**Примеры:**
```
https://infostart.ru/1c/articles/1293341/
https://infostart.ru/public/886103/
```

**Что извлекается:**
- Заголовок статьи
- Основной текст статьи (до 8000 символов)
- Автор (если доступен)

**Особенности:**
- Автоматическая очистка навигационных элементов
- Удаление блоков комментариев
- Поддержка разных типов публикаций (статьи, публикации)
